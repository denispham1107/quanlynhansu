PHẦN MỀM GIAO VIỆC NHÂN VIÊN - HTML/CSS/JS + FIREBASE

File chính:
- index.html
- styles.css
- app.js
- firebase-config.js
- firestore.rules
- firebase.json

Bước nhanh:
1. Tạo Firebase Project.
2. Bật Authentication > Email/Password.
3. Tạo Firestore Database.
4. Tạo Web App và copy config vào firebase-config.js.
5. Tạo tài khoản Admin đầu tiên trong Authentication.
6. Tạo document users/{ADMIN_UID} với role = admin.
7. Publish firestore.rules.
8. Mở index.html bằng local server hoặc deploy Firebase Hosting/GitHub Pages.
